# 📊 Sales Data Analysis using SQL

## 📌 Project Overview

In this project, I analyzed a retail sales dataset using SQL to understand business performance and extract useful insights.

## 🎯 Problem Statement

The goal of this project is to explore sales data and identify patterns related to regions, products, and customers that can help in better decision-making.

## 🛠 Tools Used

* MySQL
* Microsoft Excel

## ❓ Key Questions Explored

* Which region generates the highest sales?
* What are the top-performing products?
* Are there any loss-making products?
* Who are the most valuable customers?
* How do sales trends change over time?

## 🔍 SQL Concepts Used

* SELECT, WHERE
* GROUP BY, ORDER BY
* HAVING
* Aggregate Functions (SUM, AVG, COUNT)
* Subqueries
* Window Functions (RANK)

## 📈 Key Insights

* I found that the West region generates the highest sales, indicating strong performance in that area.
* Some products are loss-making, which could be due to pricing or cost issues.
* A small group of customers contributes a large portion of total revenue.
* Sales trends vary over time, which may indicate seasonal patterns.
* Some categories have lower profit margins, suggesting a need for improvement.
